# CRC-homelessness-simulation-code---Thesis-2026
Here you can find the R code for the Monte Carlo simulation study I implemeted for my thesis at CBS. The code applied to the real homeless data runs through the same lines of code.

## Famous people simulation

In the famous simulation folder, few model selection processes are developed with relatively complex simulated data. The inclusion probabilities vary depending on three covariates and are contradicting the population distribution for famous people.

## Simple Simulation test

In this folder, all model selections built are coded using simple simulated data. A difference is made between codes with the standard estimator and codes with the Generalized Chapman estimator (which can be found in the folder Generalized_Chapman). With the homelessness data (not accessible here), every file estimates the population size for both estimators simultaneously.
The Bayesian model NPLCM is also applied to this simulated data (NPLCM folder).
